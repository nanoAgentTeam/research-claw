# Open Research Claw 项目概览

## 这是什么

Open Research Claw 是一个面向学术论文写作的 AI Agent 系统。它让研究者通过自然语言对话来管理和撰写学术论文。

## 能做什么

- 从零创建一个论文项目，或导入已有的 Overleaf 项目
- 使用单 Agent 或多 Agent 协作来撰写、修改论文
- 自动同步到 Overleaf，用 Git 管理每一次修改的版本
- 通过 CLI 本地交互，或通过飞书 / Telegram / QQ 远程交互
- 设置定时任务自动调研，结果推送到 IM（定时功能尚未验证）

## 设计理念

系统围绕几个核心思路构建：

1. **两个空间分离**：Default（大厅）负责项目管理和聊天，Project（工作间）负责具体的论文工作。职责清晰，互不干扰。

2. **Session 隔离**：每次进入项目都创建独立的工作会话，对话历史、子 Agent 工作区互相隔离，不会串。

3. **Agent 权限分层**：主 Agent 有完整的项目读写权限，子 Agent 只能在隔离的 overlay 中工作，产出需要审阅或合并才能进入项目。

4. **自动版本控制**：每次文件修改都被 Git 追踪并自动提交，随时可以回退，不怕改坏。

5. **Overleaf 双向同步**：本地修改可以推送到 Overleaf，Overleaf 上的修改也可以拉取到本地。

## 整体架构

系统分为四层：

- **Channel 层**：CLI、Telegram、飞书、QQ（含 Gateway Web UI），负责接收和展示消息
- **Message Bus**：异步消息队列，解耦 Channel 和 Agent，支持多渠道同时工作
- **Agent Core**：AgentLoop、工具系统、上下文管理、调度器，是系统的核心
- **Reasoning 层**：LLM Provider 抽象，支持 OpenAI 兼容的 API
