# 邮箱推送配置

## 前置准备

在配置邮箱推送之前，你需要准备好以下信息：

- 一个支持 SMTP 的邮箱账号
- 邮箱的 SMTP 授权码（非登录密码）

### 常见邮箱 SMTP 信息

| 邮箱服务商 | SMTP 服务器            | 端口 | 加密方式    |
| ---------- | ---------------------- | ---- | ----------- |
| QQ 邮箱    | smtp.qq.com            | 587  | STARTTLS    |
| 163 邮箱   | smtp.163.com           | 465  | SSL         |
| Gmail      | smtp.gmail.com         | 587  | STARTTLS    |
| Outlook    | smtp-mail.outlook.com  | 587  | STARTTLS    |

### 获取 SMTP 授权码

不同邮箱获取授权码的方式不同，以下以 QQ 邮箱为例：

1. 登录 QQ 邮箱网页版，进入「设置」→「账户」
2. 找到「POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV 服务」
3. 开启「SMTP 服务」
4. 按照提示发送短信验证，获取授权码
5. 妥善保存授权码，后续配置时需要使用

> **注意**：授权码不等于邮箱登录密码，请勿混淆。

## WebUI 配置

### 1. 启动 WebUI

```bash
python cli/main.py gateway
```

### 2. 访问配置页面

访问 `http://127.0.0.1:18790/ui/`，点击「推送管理」标签页。

### 3. 添加 SMTP 配置

在「邮件推送管理」区域，点击「添加 SMTP 配置」：

- **名称**：给这个 SMTP 配置起个名字，方便识别（如：QQ 邮箱、公司邮箱）
- **服务商预设**：选择你的邮箱服务商（QQ / 163 / Gmail / Outlook / 自定义），选择后会自动填入 SMTP 服务器和端口
- **SMTP 服务器**：SMTP 服务器地址（选择预设后自动填入）
- **端口**：SMTP 端口号（选择预设后自动填入）
- **用户名**：你的邮箱地址
- **密码**：邮箱的 SMTP 授权码（不是登录密码）
- **发件人邮箱**：显示的发件人地址（一般与用户名相同）
- **发件人名称**：显示的发件人名称（如：Research Claw）
- **TLS 加密**：是否启用 TLS（推荐保持开启）

### 4. 测试 SMTP 配置

点击「测试」按钮，输入一个收件邮箱地址：

- 如果配置正确，会显示「发送成功」
- 你的收件邮箱会收到一封测试邮件

如果测试失败，请检查：

- SMTP 服务器地址和端口是否正确
- 授权码是否正确（注意不是登录密码）
- 邮箱是否已开启 SMTP 服务
- 网络是否能访问 SMTP 服务器

### 5. 添加收件人

SMTP 配置完成后，在下方的「收件人」区域，点击「添加收件人」：

- **收件邮箱**：接收推送通知的邮箱地址
- **SMTP 配置**：选择上一步添加的 SMTP 配置
- **备注**：方便识别的备注信息

### 6. 启用推送

确保收件人的状态为「已启用」，定时任务执行时就会自动发送邮件通知。

## 环境变量配置（可选）

除了 WebUI 配置外，也可以通过环境变量配置 SMTP（作为后备方案）：

```bash
export CONTEXT_BOT_SMTP_HOST="smtp.qq.com"
export CONTEXT_BOT_SMTP_PORT="587"
export CONTEXT_BOT_SMTP_USER="your_email@qq.com"
export CONTEXT_BOT_SMTP_PASS="your_smtp_auth_code"
export CONTEXT_BOT_SMTP_FROM="your_email@qq.com"
export CONTEXT_BOT_SMTP_TLS="1"
```

> WebUI 中配置的 SMTP profile 优先级高于环境变量。

## 注意事项

### 安全提示

- 授权码请妥善保管，不要泄露
- 建议使用专门的邮箱账号用于推送通知
- 密码在 API 返回中会被隐藏，仅显示是否已设置

### 常见问题

#### Q1: 收不到邮件

**可能原因**：

- SMTP 授权码错误或已失效
- 邮箱未开启 SMTP 服务
- 收件邮箱地址填写错误
- 邮件被收件方的垃圾邮件过滤器拦截
- 网络无法连接 SMTP 服务器

**排查步骤**：

1. 在 WebUI 中点击 SMTP 配置的「测试」按钮
2. 确认授权码是否正确
3. 检查邮箱的垃圾邮件文件夹
4. 查看服务器日志，确认是否有网络错误

#### Q2: 如何推送给多个收件人

**方法**：添加多个收件人配置即可。

1. 在 WebUI 的收件人区域，点击「添加收件人」
2. 填入不同的收件邮箱地址
3. 可以使用同一个 SMTP 配置，也可以使用不同的 SMTP 配置
4. 使用备注字段区分不同的收件人

#### Q3: QQ 邮箱授权码过期怎么办

QQ 邮箱的授权码可能会因为安全原因失效，需要重新生成：

1. 登录 QQ 邮箱网页版
2. 进入「设置」→「账户」
3. 重新生成授权码
4. 在 WebUI 中更新 SMTP 配置的密码字段
