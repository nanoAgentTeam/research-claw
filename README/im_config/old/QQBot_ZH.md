# QQ

QQ 是一款覆盖广泛用户群体的即时通讯平台，提供文字、语音、图片、文件等多种沟通能力，并支持群聊、频道等协作场景，适用于个人交流与团队协同。

该接入方式可与QQ Bot 进行连接，通过平台的长连接事件订阅机制接收消息与事件回调，从而在不对外暴露公网 Webhook 地址的情况下实现稳定、安全的消息收发与自动化能力集成。

参考：https://github.com/sliverp/qqbot/blob/main/README.zh.md

# 设置QQ im 聊天

## 配置QQ Bot

### 1.注册QQ开放平台

前往腾讯QQ开放平台官网，默认无法使用您的QQ账号直接登录，需要新注册QQ开放平台账号。

https://q.qq.com/#/

![1772516081634](image/QQBot/1772516081634.png)
首次注册之后，可以按照QQ开放平台的指引设置超级管理员。

![1772516239878](image/QQBot/1772516239878.png)

手机QQ扫码成功后，继续下一步填写主体相关信息。

此处以“个人”为例，按照指引依次输入姓名、身份证号、手机号、验证码，点击继续进入下一步人脸认证。
![1772516256090](https://file+.vscode-resource.vscode-cdn.net/Users/alucard/Code/open-overleaf-claw-260227/open-overleaf-claw/README/%E8%B4%A6%E5%8F%B7%E9%85%8D%E7%BD%AE/im%E9%85%8D%E7%BD%AE/image/QQBot/1772516256090.png)

使用手机QQ扫码进行人脸认证。

![1772516315227](image/QQBot/1772516315227.png)

人脸识别审核通过后，即可登录进入QQ开放平台。
![1772516327601](image/QQBot/1772516327601.png)

### 2.创建一个QQBot机器人

在QQ开放平台的QQ机器人页面，可以创建机器人。
![1772516346532](image/QQBot/1772516346532.png)
![1772516356381](image/QQBot/1772516356381.png)

QQ机器人创建完成之后，可选择机器人点击进入管理页面。
![1772516369900](image/QQBot/1772516369900.png)

在QQ机器人管理页面获取当前机器人的AppID和AppSecret，复制并将其保存到个人记事本或备忘录中（请注意数据安全，勿泄露），后续在“步骤3中配置OpenClaw“中需要使用。

注意：出于安全考虑，QQ机器人的AppSecret不支持明文保存，首次查看或忘记AppSecret需要重新生成。
![1772516410242](image/QQBot/1772516410242.png)
![1772516421099](image/QQBot/1772516421099.png)

### 3.沙箱配置

在QQ机器人的“开发管理”页面，在“沙箱配置”中，设置单独聊天（选择“在消息列表配置”）。

您可以按照自己的使用场景进行配置，也可以完成后续步骤之后再回到本步骤进行操作。

⚠️ 注意：
此处已创建的QQ机器人无需进行发布上架对所有QQ用户公开使用，在开发者私人的（沙箱）调试下使用体验即可。
QQ开放平台不支持机器人的“在QQ群配置”操作，只支持单独和QQ机器人聊天。
![1772516435649](image/QQBot/1772516435649.png)

注意：选择“在消息列表配置”时，需要先添加成员，再通过该成员的QQ扫码来添加机器人。
![1772516450100](image/QQBot/1772516450100.png)
此处注意添加成员成功之后，还需要使用QQ扫码添加

![1772516465600](image/QQBot/1772516465600.png)
此时您的QQ账号添加机器人之后，还不能与该机器人正常进行对话，会提示“该机器人去火星了，稍后再试吧”，因为QQ机器人此时尚未与OpenClaw应用打通。

您需要继续后面的步骤，为OpenClaw应用配置QQ机器人的AppID和AppSecret。

![1772516524342](image/QQBot/1772516524342.png)

（可选）您也可以参考前述步骤添加更多成员：首先在成员管理页面中添加新成员，然后在沙箱配置页面中添加成员，之后新成员即可通过QQ扫码添加该QQ机器人。
![1772516554121](image/QQBot/1772516554121.png)
![1772516566218](image/QQBot/1772516566218.png)
![1772516577359](image/QQBot/1772516577359.png)
